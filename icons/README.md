# 图标文件说明

请在此文件夹中添加以下图标文件：

- `icon16.png` - 16x16 像素图标
- `icon48.png` - 48x48 像素图标  
- `icon128.png` - 128x128 像素图标

## 获取图标

你可以：
1. 使用在线图标生成工具（如 https://www.favicon-generator.org/）
2. 使用设计软件（如 Photoshop、Figma）制作
3. 使用免费图标资源（如 Icons8、Flaticon）

图标应该是 PNG 格式，支持透明背景。

## 临时解决方案

如果暂时没有图标，你可以创建简单的占位图标，或者使用纯色图片作为临时图标。
